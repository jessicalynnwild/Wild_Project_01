This repository will be used for Project_01.  All relevant material will be organized in 7 separate folders.  

File | Description
---|---------------------------------------------------------------------
Admin | contains all admin information files
Background | contains the background information for the analysis
Code | contains all R scripts for this project
Data | contains the processed data used for analysis
DataRaw | contains all raw data
Dissemination | contain all abstract, poster, and manuscript files
Reports | contains all output, rmarkdown files and reports

This project investigates the titanic dataset.  Descriptive analysis and plots are generated relating to class, age, sex, and survival status on a sample of passengers.
